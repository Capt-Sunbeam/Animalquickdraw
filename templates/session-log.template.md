## Template Usage Notes

**When creating a session log:**

1. Fill in Session Overview with user-provided date/time
2. Document objectives from session start (or WHERE_WE_ARE)
3. List all accomplishments with file paths
4. Document any decisions with full context
5. Record all issues and their resolutions
6. Include test results even if no new tests
7. Be specific about where work stopped
8. Add helpful notes for next session
9. Fill in metrics for tracking

**Important:**
- Create this log at the END of every session
- Use actual file paths, not placeholders
- Be specific enough that a fresh AI can resume
- Reference Decision Log entries if significant decisions were made
