## Template Usage

When creating WHERE_WE_ARE for a new project:

1. Replace `[Project Name]` with your project name
2. Fill in Current Status after skeleton is complete
3. Update Quick Links with actual file paths
4. Add slices to appropriate tables as they're planned/completed
5. Update after every session following Session End Workflow

**Update Frequency:**
- Total Sessions counter: Increment every session
- Current Status: Every session
- Session History: Add entry every session (never remove old entries)
- Next Steps: Every session
- Progress tables: When slices start/complete
